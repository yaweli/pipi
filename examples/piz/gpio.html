to make the red light on the pi flash every second
bash script code example: 
#!/bin/sh
cd /sys/class/gpio
echo "14" > unexport
echo "14" > export
cd gpio14
echo out > direction
sleep 1
while true
do
        echo "1" > value
        sleep 1
        echo "0" > value
        sleep 1
done

if you start it on pi startup (using /etc/rc.local) the pi will flash the light forever
